# hackernews-scrape

Scrape [Hacker News](https://news.ycombinator.com/) data into postgresql database. Python script idea derived from [ashish01/hn-data-dumps](https://github.com/ashish01/hn-data-dumps).

## Setup

```bash
cp database.ini.example database.ini # change db config if necessary
docker-compose up -d
pip3 install -r requirements.txt
python3 hackernews-scrape.py
```

### Sometime CAGG is not as fast as direct index

The CAGG doesn't reduce enough cardinality from the base table.

Time needed to create CAGG

```
hackernews=# CREATE MATERIALIZED VIEW daily_activity
WITH (timescaledb.continuous) AS
SELECT "by",
   time_bucket(INTERVAL '1 day', time) AS bucket,
   COUNT(*) as post_count,
   SUM(score) as score_sum
FROM items
GROUP BY "by", bucket
WITH NO DATA;
CREATE MATERIALIZED VIEW
Time: 310.361 ms
hackernews=# CALL refresh_continuous_aggregate('daily_activity', NULL, NULL);
```
